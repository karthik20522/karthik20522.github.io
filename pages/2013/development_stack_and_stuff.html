Development Stack and stuff - 2013 Edition

Dec, 2013

For the past year or two, I had been dabbling with different technologies, frameworks looking for an ideal combination of frontend, backend and development tools. Following are what I tend to use and recommend for both personal and consultancy projects.
I usually customize most of these open-source projects as per my needs

Project Management
  • Asana - https://asana.com
Database
  • MongoDB
CMS
  • Calipso - http://calip.so/
Bulletin Board
  • NodeBB - http://nodebb.org/
Configuration Management
  • Etcd - https://github.com/coreos/etcd
Logging and Analysis
  • Logstash - http://logstash.net
  • Kibana - http://www.elasticsearch.org/overview/kibana
  • Log.io - http://logio.org
Search
  • ElasticSearch - http://www.elasticsearch.org
  • Algolia - http://www.algolia.com
Source Control
  • Github - http://github.com
AWS - Amazon Web Services
  • SQS - Simple Queue service
  • SES - Simple Email Service
Hosting
  • Rackspace / Amazon
  • Digital Ocean - https://www.digitalocean.com
Image processing
  • GraphicsMagick with Node API - http://aheckmann.github.io/gm/
Development Tools/Frameworks/Languages
  • .NET
    • ASP.NET MVC
    • ASP.NET WebAPI for REST services - http://www.asp.net/web-api
    • Dapper.NET or Simple.Data for SQL ORM's and MongoDB CSharp Driver for MongoDB access
    • FluentSecurity for user authentication and roles http://www.fluentsecurity.net/
  • NodeJS
    • NodeProxy for proxy and load balancing service
    • Node Cluster for Multi Node process
    • Node Express Boilerplate - https://github.com/mape/node-express-boilerplate
Misc Libraries and websites
  • ASP.NET Razor Helpers - http://www.mikepope.com/blog/documents/WebHelpersAPI.html
  • OverAPI - http://overapi.com/
  • JQuery Plugins - http://www.unheap.com/
  • MicroJs - http://microjs.com/
